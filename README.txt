Example scripts for working with Microsoft Active Directory using Python and the LDAP3 module (http://ldap3.readthedocs.io/installation.html)

Project Files:

ad_config.json contains the values of the AD account to use when querying, domain controller names, and search paths 

ad_common_tools.py loads configuration and common functions used in all scripts

ou_group_membership.py demonstrates how to pull and change an AD group's membership

ou_groups_list.py pulls a list of AD groups in a specific department OU
